SathSafar - Demo Flutter App (Ready)
===================================

This is a demo build of the SathSafar Flutter app with OTP demo login (no Firebase).
Details:
- App name: SathSafar
- Demo login: mobile 9999999999 (auto-verify) or any phone with OTP 123456
- Theme: Purple gradient (logo colors)
- Includes UI for Wallet, Recharge, PAN, Transactions, Support, Profile (all demo placeholders)

How to run:
1. Install Flutter SDK (https://flutter.dev)
2. Unzip this project and open in VS Code or Android Studio.
3. Run: flutter pub get
4. Run on device/emulator: flutter run
5. To build release APK: flutter build apk --release
